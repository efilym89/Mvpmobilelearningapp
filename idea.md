# Идея приложения для обучения персонала

## Описание

Разрабатывается мобильное приложение для обучения персонала. Приложение должно работать на гаджетах под управлением iOS и Android.

## Основная задача

Основная задача приложения — предоставлять пользователю учебные материалы и принимать экзамены на их основе.

## Базовый сценарий использования

Пользователь изучает материалы в приложении, после чего проходит экзамен для проверки усвоения знаний.
